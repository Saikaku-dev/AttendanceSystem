# 出席管理システム（PHP練習用）

このシステムは、**PHPの練習**を主な目的として作成された簡単な出席管理システムです。学校の出席確認システムを参考にしています。

## 機能

- 遅刻、早退、欠席などの状況を記録
- 日付や時間帯の選択
- 理由の記入欄を提供

## インストール方法

1. リポジトリをクローン
2. ファイルをWebサーバーのディレクトリに配置
3. `attendance.php` にアクセスして利用開始

## 使い方

1. フォームで欠席の種類を選択
2. 日付範囲を選択
3. 関連する時間帯を選択
4. 理由を入力
5. フォームを送信

> このシステムは、PHPの学習を目的としており、学校の出席確認システムを参考にしています。
